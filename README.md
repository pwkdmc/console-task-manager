# Console task manager
A collection of console utility programs written in C++.
## Features
1. **Random Password Generator** – Generate passwords with customizable length (6–32 chars) and character types (digits, uppercase/lowercase letters, special chars). Includes strength analysis.
2. **Number System Converter** – Convert numbers between bases 2–16. Supports negative numbers and digits A–F.
3. **ATM Simulator** – Simple bank account simulation with withdrawal and deposit functions. Tracks operation limits per session.
4. **Rock–Paper–Scissors Game** – Play against the computer with three difficulty levels (easy, medium, hard). Tracks scores and displays statistics.
5. **Text Analyzer** – Analyzes input text (max 1000 chars). Shows character/word/sentence count, longest/shortest words, letter frequency, and palindrome detection.
6. **Countdown Timer** – Single or interval timers with pause/resume, cancel, and percentage-based notifications (50%, 25%, 10%).
